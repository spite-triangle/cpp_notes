# 简介

# 基本概念

- `SSM`: 由 `Spring framework`、`SpringMVC`、`Mybatis` 三个框架构成, 是经典的 `jave web` 的开发模式
  - `Spring framework`: 核心框架，提供了依赖注入（DI）和面向切面编程（AOP）等功能
  - `SpringMVC`: 基于`MVC` 模式处理 Web 请求和响应，是 `Spring framework` 中的一个模块
  - `Mybatis`: 一个持久层框架，简化了数据库操作
- `Tomcat`: 用于部署和运行 `Java Web` 应用，即 `war` 包
- `Spring Boot`: 是基于 `Spring framework` 的快速开发工具，提供了自动配置、嵌入式服务器等功能，**简化了 `Spring framework` 的开发过程，但并非原有功能增强**

# Spring

> [!tip]
> 由于 `Spring framework` 是核心框架，通常会简称为 `Sring` ，但叫法上就和 `Spring` 系列存在混淆，注意区分。

[Spring](https://spring.io/) : 一个开源的 Java 平台，包含一系列的[产品](https://spring.io/projects)，用于构建企业级应用
- `Spring framework`: 核心框架，提供了依赖注入（DI）和面向切面编程（AOP）等功能
- `Spring Boot`: 简化了 `Spring framework` 的开发过程，但并非原有功能增强
- `Spring Cloud`: 提供了一系列的工具和服务，用于构建分布式系统
